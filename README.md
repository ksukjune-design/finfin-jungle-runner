# 🚲 핀핀 정글 등원 대작전

핀핀(FINFIN) 밸런스바이크 PRO V2 홍보 동영상을 인트로로 사용하는 **모바일 원버튼 러닝 게임** (세로·가로 화면 모두 지원, 플레이 중 회전 가능).
"등원은 재밌쟁" 머리띠를 한 시바견이 페달 없는 핑크 핀핀 밸런스바이크를 타고,
정글 속에서 호랑이에게 쫓기며 유치원까지 달립니다.

## 🎮 지금 플레이

**https://ksukjune-design.github.io/finfin-jungle-runner/**

정적 파일만으로 동작하는 인스턴스형 서비스입니다 (서버·DB 없음, GitHub Pages CDN 배포).
휴대폰 브라우저에서 위 주소로 접속하면 바로 플레이됩니다.

## 로컬 실행 방법

```bash
node tools/server.js
# → http://localhost:8080 접속 (모바일이면 같은 Wi-Fi에서 PC IP:8080)
```

정적 파일만 사용하므로 아무 웹서버에나 올려도 됩니다. (mp4 재생을 위해 Range 요청을 지원하는 서버 권장 — `tools/server.js`는 지원)

## 게임 방법

| 조작 | 동작 |
|---|---|
| 오른쪽 탭 | 점프 (공중에서 한 번 더 = 2단 점프) |
| 왼쪽 꾹 누르기 | 슬라이드 (낮은 나뭇가지·투칸 밑 통과) |
| 공중에서 왼쪽 꾹 | 내려찍기(빠른 착지) |
| (PC) Space/↑ = 점프 · ↓/S = 슬라이드 · P/Esc = 일시정지 | |

### 10 스테이지 여정 (완주형)

- **10,000m를 완주하면 유치원 도착!** 1,000m마다 완전히 다른 스테이지 10개:
  🌿정글 → 💧폭포강가 → 🏖️해변 → 🎋대나무숲 → 🏡시골마을 → 🌇노을들판 →
  🌙반딧불숲 → 🏜️붉은협곡 → 🏙️도시거리 → 🏫유치원길(벚꽃 피날레).
- 스테이지마다 **배경·장애물 구성·아이템·앰비언트(낙엽/안개/반딧불/꽃잎)·BGM이 전부 다름**.
- 스테이지 클리어 시 **폭죽 세리머니 + 진행 도트 + 다음 스테이지 미리보기 + 골드 퍼크 카드** 이벤트.
- **체크포인트**: 클리어한 스테이지부터 다음 판 이어 등원 가능. 완주 시 전용 엔딩.
- **BGM 10곡**: 스테이지별로 다른 프로시저럴 칩튠 (WebAudio 합성 — 외부 음원 없음, 저작권 프리).
- 상단 여정 진행 바(🏫)로 남은 거리 표시. 전체 게임 속도 1.15배.

### 로그라이크 시스템 (DX)

- **🚲 바이크 차고**: 도전과제로 해금되는 5종 바이크 — 핑크(기본), 로켓(3단 점프),
  로우라이더(가지·새 밑 자동 통과), 날개(구덩이 활공), 탱크(새 튕겨냄·피격 반감·느림).
- **🎁 퍼크 카드**: ~900m마다 시바견과 호랑이가 서로 약올리는 카툰 컷과 함께
  퍼크 3장 중 1장 선택 — 뼈 1.5배, 상시 자석, 니어미스 2배 등 9종으로 런 빌드 구성.
- **스테이지 전용 장애물**: 뱀·벌떼(공통), 진흙·원숭이 코코넛 투척(아침/노을),
  구르는 바위(노을/밤), 도깨비불(밤) + 랜덤 소나기, 중경 실루엣 패럴랙스, 낙엽/꽃가루/반딧불.

### 재미 요소 (엔드유저 리서치 기반 개편)

- **패턴 레벨디자인**: 난이도 밴드(5단계)별 청크 풀에서 가중 추첨 — 첫 300m는 학습 구간, 2500m+ 전문가 패턴. 같은 패턴 연속 등장 방지.
- **콤보 & 피버**: 뼈다귀 연속 수집 시 콤보 보너스, 게이지가 차면 🌈 **피버 타임** — 무적 질주 + 자석 + 점수 2배.
- **파워업**: ⛑️ 안전모(1회 보호 — 헬멧 쓰면 안전!), 🧲 자석, 🚀 부스터, 💛 황금 뼈다귀.
- **니어미스 보너스**: 장애물을 아슬아슬하게 스치면 +30 "아슬아슬!"
- **스테이지 순환**: 700m마다 🌞 아침 → 🌇 노을 → 🌙 밤(반딧불) 정글로 전환.
- **부활**: 잡히기 직전 한 판에 한 번 "이어서 달리기".
- **도전과제 12종** + 기록 공유(📣 자랑하기), 일시정지, 사운드 토글.
- 튜토리얼: 첫 판은 쉬운 배치 + 조작 말풍선 안내.
- 세로·가로 화면 모두 지원, 플레이 중 회전해도 이어서 달립니다.

## 에셋 재생성 (Gemini 이미지 생성 파이프라인)

`tools/` 폴더에 생성 파이프라인이 있습니다. 영상 프레임(`assets/ref/`)을
레퍼런스로 넣어 캐릭터 일관성을 유지하고, 크로마키(초록/마젠타) 배경으로 생성 후
투명 PNG로 후처리합니다.

```bash
export GEMINI_API_KEY=<키>
# 생성: node tools/gen.js <출력.png> <종횡비> <프롬프트파일> [레퍼런스이미지...]
node tools/gen.js tools/out/shiba_sheet.png 1:1 tools/prompts/shiba_sheet.txt assets/ref/ref_full_shiba.jpg
# 후처리(크로마키+슬라이스+트림): KEY_COLOR=green|magenta
KEY_COLOR=green node tools/process.js slice tools/out/shiba_sheet.png assets/sprites/shiba_ 2 2 420
```

프롬프트는 `tools/prompts/*.txt` 에 있으며, API 키는 코드에 하드코딩하지 말고
환경변수로만 전달하세요.
